4、常態性(Normality)不滿足

  • 散佈圖特徵：適用任何形狀的散佈圖，使用者可根據數據特點和問題自行定義。
  • 建議應用場景：當數據無法簡單歸納為一階或二階模型時，使用者可根據自身領域知識與經驗自訂模型。

以下特徵:非隨機分布的殘差, 離散度隨預測值變化而變化, 有清晰的曲線或模式) (numerical variable):

解決方法：

  1. 考慮增加自變數的多項式項目:需要添加更多的特征來捕捉數據中的非線性關系，或者使用非線性迴歸模型

2. if 違反獨立性

解決方法：

  1. 將數據按照均值為0、標準差為1進行線性變換，使得數據具有相似的尺度和變異性
  2. Normalization：將數據縮放到特定的範圍內，通常是[0, 1]或[-1, 1]，使得數據具有相似的尺度。歸一化可以減少數據的尺度差異
  3. Logarithmic Transformation：對數據取對數，可以將數據的分佈形狀進行調整，使其更加接近正態分佈。對數轉換通常用於處理偏態分佈的數據或者減少數據的離群值對分析結果的影響。
  4. Power Transformation：將數據進行冪次轉換，例如平方、立方或開方等，可以調整數據的分佈形狀，使其更加符合正態分佈或者減少數據的異方差性 /li>
  5. Box-Cox變換：Box-Cox變換是一種廣泛應用的冪函數變換方法，通過估計一個參數λ來確定進行何種程度的冪函數轉換。Box-Cox變換可以使數據更加接近正態分佈。

3. if 違反同質變異性

解決方法：

  1. 可使用異方差穩健迴歸的方法（先用white test 來做假設檢定，然後再做OLS估計）
  2. Variance Stabilizing Transformation：通過數據轉換，使得不同組的變異性更加接近。例如，對數轉換、平方根轉換等轉換方法可以穩定變異性，使其更符合同質變異性的假設。
  3. Weighted Analysis of Variance：對於同質變異性的假設不成立的情況，可以使用加權方差分析方法，對不同組的變異性進行加權處理，以考慮不同組的變異性差異。

4. if 產生多重共線性的問題：

解決方法：

  1. 減少相關自變數的數量
  2. 使用主成分分析: 將原始的自變數轉換為一組無關的主成分，這些主成分是原始變量的線性組合，彼此之間不存在相關性。然後，可以僅使用與目標變量高度相關的主成分作為模型的自變數。

5. if 有離群值：

解決方法：

  1. 考慮移除或是調整離群值

6. if 線性假設和常態性假設同時違反

解決方法：

  1. 考慮增加自變數的多項式項目以處理線性假設的違反且同時對應變數或自變數進行轉換，例如對數轉換，解決兩種假設問題

7. if 線性假設和同質變異性假設同時違反

解決方法：

  1. 考慮增加自變數的多項式項目以處理線性假設的違反且使用異方差穩健迴歸方法來處理同質變異性的問題

8. if 線性假設、常態性假設和同質變異性假設同時違反

解決方法：

  1. 考慮增加自變數的多項式項目以處理線性假設的違反
  2. 對應變數或自變數進行轉換，例如對數轉換
  3. 使用異方差穩健迴歸方法來處理同質變異性的問題

特性：

  • 適合處理高維度的數據

殘差圖可能違反的一些假設以及對應的特徵：

1.同方差性假設（Homoscedasticity）：

  • 殘差圖中殘差的方差應該在各個預測值或者自變數值的範圍內保持穩定。當殘差的方差隨著預測值或者自變數值的增大而增大或者減小時，就會違反同方差性假設。這可能表明模型存在異方差性，需要進行變換或者使用合適的模型。

2.線性性假設（Linearity）：

  • 殘差圖應該沒有明顯的趨勢或者模式，而應該呈現隨機的分佈。如果殘差圖中出現了明顯的非線性模式，可能表明模型對數據的擬合不夠好，需要考慮添加更多的特徵或者使用非線性模型。

3.正態性假設（Normality）：

  • 殘差應該呈現接近正態分佈的形態，即殘差圖中的點應該均勻地分佈在零點周圍，形成一條近似水平的直線。如果殘差圖中的點呈現明顯的偏離或者不規則的分佈，可能表明模型存在正態性假設的問題，需要進行變換或者使用合適的統計檢驗來確認。

4. 獨立性假設（Independence）：

  • 殘差之間應該是獨立的，即殘差不應該呈現出時間序列相關性或者空間相關性。如果殘差圖中出現了明顯的自相關結構，可能表明模型存在獨立性假設的問題，需要進一步檢查和處理。

QQ plot的判定

1.正態性假設（Normality）：

QQ 圖通常用於檢驗數據是否符合正態分佈。如果 QQ 圖偏離了直線，可能表明數據的分佈與正態分佈不符合。

特徵包括:

  • 尾部偏差：QQ 圖尾部出現彎曲，可能表示數據分佈的尾部比正態分佈更厚或更薄。
  • 中心偏差：QQ 圖中心部分的點偏離了直線，可能表示數據在中心區域與正態分佈有所不同。

2.同方差性假設（Homoscedasticity）：

QQ 圖也可以用於檢驗數據的同方差性。如果 QQ 圖的點在一條直線附近均勻分佈，可能表明數據的同方差性得到了滿足。

特徵包括:

  • 扇形模式：QQ 圖中點圍繞着中心線呈現出扇形，可能表示方差隨着變量值的變化而變化，違反了同方差性假設。

3.獨立性假設（Independence）：

雖然 QQ 圖主要用於檢驗數據分佈的形態，但在某些情況下，它也可以提供數據間是否相互獨立的線索。如果 QQ 圖中點呈現出明顯的模式或趨勢，可能暗示數據間存在一定的相關性或時間序列結構。

4.分佈假設：

QQ 圖還可以用於比較數據的分佈與其他理論分佈（如指數分佈、Weibull 分佈等）之間的一致性。當 QQ 圖與理論分佈的偏離超出預期範圍時，可能需要重新評估所選擇的分佈模。
