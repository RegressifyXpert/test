專有名詞解析——殘差頁面

April 30, 2024

迴歸必須滿足的四個假設：

  1. 線性關係
  2. 獨立性
  3. 常態性
  4. 同質變異性

關於違反假設的解決方法：

1. if 違反線性關係(以下特徵:非隨機分布的殘差, 離散度隨預測值變化而變化, 有清晰的曲線或模式) (numerical variable):

解決方法：

  1. 考慮增加自變數的多項式項目:(需要添加更多的特征來捕捉數據中的非線性關系。)，或者使用非線性迴歸模型

2. if 違反獨立性

解決方法：

  1. 將數據按照均值為0、標準差為1進行線性變換，使得數據具有相似的尺度和變異性
  2. Normalization：將數據縮放到特定的範圍內，通常是[0, 1]或[-1, 1]，使得數據具有相似的尺度。歸一化可以減少數據的尺度差異
  3. Logarithmic Transformation：對數據取對數，可以將數據的分佈形狀進行調整，使其更加接近正態分佈。對數轉換通常用於處理偏態分佈的數據或者減少數據的離群值對分析結果的影響。
  4. Power Transformation：將數據進行冪次轉換，例如平方、立方或開方等，可以調整數據的分佈形狀，使其更加符合正態分佈或者減少數據的異方差性 /li>
  5. Box-Cox變換：Box-Cox變換是一種廣泛應用的冪函數變換方法，通過估計一個參數λ來確定進行何種程度的冪函數轉換。Box-Cox變換可以使數據更加接近正態分佈。

3. if 違反同質變異性

解決方法：

  1. 可使用異方差穩健迴歸的方法（先用white test 來做假設檢定，然後再做OLS估計）
  2. Variance Stabilizing Transformation：通過數據轉換，使得不同組的變異性更加接近。例如，對數轉換、平方根轉換等轉換方法可以穩定變異性，使其更符合同質變異性的假設。
  3. Weighted Analysis of Variance：對於同質變異性的假設不成立的情況，可以使用加權方差分析方法，對不同組的變異性進行加權處理，以考慮不同組的變異性差異。
