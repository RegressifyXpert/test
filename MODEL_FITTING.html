專有名詞解析——模型配適頁面

April 30, 2024

一、基本介紹

1、自變數 (independent variable):

在統計模型中，影響或解釋應變數變化的變數；即Ｘ值。

2、應變數 (dependent variable):

在統計模型中，受自變數影響或解釋的變數，也稱為反應變數；即Ｙ值。

3、交互作用項 (interaction):

自變數之間相互影響的效應，即當一個自變數的效應受到另一個自變數的影響時。

4、虛擬變數 (dummy variable):

在統計建模中，用於表示類別型變數的二元變數，通常用0和1表示不同類別。

[例]有一類別變數color中含有red, blue, yellow三種資料，利用虛擬變數轉換後得到三個連續變量color_red, color_blue, color_yellow，且當資料為紅色時，color_red=1, color_blue=0, color_yellow=0 ，以此類推。

5、殘差(residual):

指實際觀測值和模型預測值之間的差異。

6、普通最小平方法 (OLS regression):

一種迴歸分析的方法，用於建立自變數對應變數的線性關係，通常用於連續性變數；在OLS中，會試圖找到一條最佳擬合的直線或超平面，使得觀察到的數據點和此之間的殘差平方和最小。

二、常見模型介紹

1、一階模型:

最基本的線性回歸模型，包含了截距項和自變量的一次項，且其假設因變量（Y）與自變量（X）之間存在線性關係。

r($ Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \ldots + \beta_n X_n + \varepsilon $$

2、二階模型:

在基本線性回歸模型上擴展，添加了自變量的平方項。

r"{$ Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \ldots + \beta_n X_n + \beta_{n+1} {X_1}^2 + \beta_{n+2} {X_2}^2 + \ldots + \beta_{2n} {X_n}^2 + \varepsilon $}"

    其中
  • {$Y$} 為應變數(dependent variable)
  • $\beta_0 $ 為截距項
  • ${\beta_1, \beta_2, \ldots, \beta_{2n+2} \ldots }$ 為自變數 $X_1 \ldots X_n ,{X_1}^2 \ldots {X_n}^2, X_1 X_2 , X_1 X_3 \ldots$ 的迴歸係數
  • $X_1 \ldots X_n ,{X_1}^2 \ldots {X_n}^2 $ 為自變數(independent variables)
  • $ X_1 X_2 , X_1 X_3 \ldots $ 為交互作用項(interaction terms)
  • $\varepsilon $ 為標準誤差項(error term)

標準誤差項 $ \varepsilon $ 的四大假設:

  1. 誤差項 $ \varepsilon $ 的平均數為0,即 $ E(\varepsilon) = 0 $.
  2. 誤差項 $\varepsilon $ 的方差恆定, 即 $ Var(\varepsilon) = \sigma^2 $.
  3. 誤差項 $ \varepsilon $ 呈現常態分佈
  4. 誤差項之間須相互獨立

上述假設對於使用迴歸分析進行統計推論十分重要，若不滿足可能需對資料進行轉換，請參閱"問題解決"頁面。

3、一階交互作用模型:

擴展一階線性回歸模型，包含了自變量的一次項以及它們之間的交互作用項。

$$ Y = \beta_0 + \beta_1 X_1 + \ldots + \beta_n X_n + \beta_{n+1} X_1 X_2 + \beta_{n+2} X_1 X_3 + \ldots + \varepsilon $$

4、二階交互作用模型:

擴展二階線性回歸模型，包含了自變量的一次項、二次項以及它們之間的交互作用項。

$$ Y = \beta_0 + \beta_1 X_1 + \ldots + \beta_n X_n + \beta_{n+1} {X_1}^2 + \ldots + \beta_{2n} {X_n}^2 + \beta_{2n+1} X_1 X_2 + \beta_{2n+2} X_1 X_3 + \ldots +\varepsilon $$

其中

  • Y 為應變數(dependent variable)
  • β0 為截距項
  • $\{ β1, β2, &ldots; , β{2n+2} &ldots; \}$ 為自變數 $X_1 &ldots; X_n ,{X_1}^2 &ldots; {X_n}^2, X_1 X_2 , X_1 X_3 &ldots; $ 的迴歸係數
  • $X1 &ldots; Xn ,{X1}^2 &ldots; {Xn}^2 $ 為自變數(independent variables)
  • $X1 &ldots; Xn ,{X1}^2 &ldots; {Xn}^2 $ 為交互作用項(interaction terms)
  • ϵ 為標準誤差項(error term)

標準誤差項 ϵ 的四大假設:

  1. 誤差項 ϵ 的平均數為0，即 E(ϵ) = 0
  2. 誤差項 ϵ 的方差恆定，即 Var(ϵ) =σ2$
  3. 誤差項 ϵ 呈現常態分佈
  4. 誤差項 ϵ 之間須相互獨立

上述假設對於使用迴歸分析進行統計推論十分重要，若不滿足可能需對資料進行轉換，請參閱"問題解決"頁面。

5、其他模型:

可供使用者自主設計更多複雜的特徵和交互作用，而這些模型可能需要更多的參數來估計，並可能更難解釋。
